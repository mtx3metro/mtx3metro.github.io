This application displays predicted train arrival times on Washington DC's WMATA Metro system and has been designed primarily for use on a hand held devices.

Individual Metro stations can be viewed by scrolling up or down on the station display.

Metro lines maybe changed by swiping the line selector at the bottom of the page.

Rotating device to landscape mode will display WMATA Metro map.

This project has been developed (almost) entirely using the TypeScript scripting language.

It uses features that are typically only available inadvanced browsers and has been tested with latest versions of Safari and Chrome.
